```py
3 > 2 and 4 > 3    # True
```
- `and` --> Devuelve `True` si ambos elementos son verdaderos.

```py
3 > 2 or 4 > 5    # True
```
- `or` --> Devuelve `True` si alguno de los elementos es verdadero.

```py
not 3 > 2        # False
```
- `not` --> Revierte el resultado, si es verdadero devuelve `False` y si es falso devuelve `True`.
