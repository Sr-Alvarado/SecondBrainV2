Se usa la etiqueta `<title>` dentro del elemento `<head>`[^1]. Este elemento es usado por los buscadores para indexar tu página web[^8].
```HTML 
<head>
  <meta charset="UTF-8">
  <title>Titulo que el navegador mostrará en la pestaña</title>
</head>
```
