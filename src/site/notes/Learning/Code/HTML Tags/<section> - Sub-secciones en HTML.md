La etiqueta `<section>` define sub-secciones que se enfocan en temas específicos dentro de la sección principal `<main>`[^1].
```HTML 
<main>
  <h1>Todo sobre los gatos</h1>
  
  <section>
    <h2>Orígen de los gatos</h2>
  </section>
</main>
```
