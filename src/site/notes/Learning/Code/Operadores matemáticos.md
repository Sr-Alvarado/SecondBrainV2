## Operaciones
- **Sum**: 3`+`2
- **Subtraction**: 3`-`2
- **Multiply**: 3`*`2
- **Division**: 3`/`2 
- **Exponente**: 3`**`2 ==Se lee: 3 al exponente 2==
- **[[Learning/Code/Módulo - Resto de una división\|Módulo]]**: 3`%`2 ==Devuelve el residuo de una división==
- **Resto**: 3`//`2 ==Devuelve solo la parte entera de una división==

## Operadores
- **Igual**: ` '==` 
- **Diferente**: ` !=` 
- **Mayor que**: ` >`
- **Menor que**: ` <`
- **Mayor o igual**: ` >=`
- **Menor o igual**: ` <=`
