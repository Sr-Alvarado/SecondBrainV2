- **`text-align:` - Alinear un texto**. Propiedad que permite alinear un texto a la derecha, centro o izquierda especificando en el valor[^1].
   ```HTML 
   <style>
      h1, h2, h3 {
        text-align: center;
      }
      h5, h6 {
        text-align: right;
      }
      p {
        text-align: left;
      }
    </style>
   ```
